# na-trave

The application was developed to allow users to mark predictions for the 2022 World Cup games, being saved in a database, which can be consulted and changed.

##

- React.JS (Vite)
= React Router DOM
- Tailwind CSS
- TypeScript
- Date-fns
- Formik
- Yup
- Planetscale
- Koa.JS
- Prisma
- TS Node Dev
- Json Web Token
- Bcrypt
